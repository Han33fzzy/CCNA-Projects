# CCNA-Projects
A collection of hands-on networking labs built in Cisco Packet Tracer while studying for the CCNA. Each folder is a self contained lab covering a specific topic, with configurations, topology diagrams and verification output.

## Skills demonstrated 
VLANS & Trunking (VTP) . Spanning Tree Protocol . Static and Dynamic Routing (RIP, EIGRP, OSPF) . Access Control List . DHCP . Troubleshooting 

## PROJECTS
- [VLANs & Trunking](01-vlans-trunking) — VLAN segmentation and trunk links between switches
- [Static Routing](02-static-routing) — Manual route configuration
- [RIP](03-rip) — Dynamic routing with RIPv2
- [DHCP](04-dhcp) — DHCP pool and relay configuration
- [OSPF](05-ospf) — Multi-area OSPF configuration
- [EIGRP](06-eigrp) — EIGRP configuration and verification
- [Spanning Tree Protocol](07-stp) — Loop prevention with STP
- [Access Control Lists](08-acls) — Standard and extended ACLs
